# features:
- Extract Media:
  - [x] tiktok
  - [] twitter
  - [] reddit
  - [] facebook
  - [] instagram
  - [] twitch (clip)
  - [] youtube
- [] `/transcribe message` transcribe voice message (OpenAi Whisper)

# usage
Install dependencies:
```bash
bun install
```

Run:
```bash
bun run index.ts
```
